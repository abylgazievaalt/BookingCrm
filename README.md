# "BookingCRM" Python project
 Project "BookingCRM" backend
## **Deploy.**
1 Download project
```
git clone url
```
2 Install docker
```
https://docs.docker.com/get-docker/
```
3 Run command
```
docker-compose up --build
```
4 Enjoy
